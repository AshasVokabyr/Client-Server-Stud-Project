# 🛒 Интернет-магазин 

Учебный проект интернет-магазина электроники. Реализован каталог товаров с фильтрацией, корзина с сохранением состояния в `localStorage` и авторизация пользователей.

## ✨ Функционал
- 📦 **Каталог товаров** с фильтрацией по категории и цене
- 🛒 **Корзина**: добавление, удаление, полная очистка
- 💾 **LocalStorage**: автосохранение корзины, данные не теряются при перезагрузке страницы
- 🧮 **Подсчет итоговой суммы** в реальном времени
- Авторизация пользователей
- 🌐 HTML/CSS/JS/PHP
