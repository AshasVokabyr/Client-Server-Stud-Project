# 🛒 Интернет-магазин (Практическая работа №4)

Учебный проект интернет-магазина электроники. Реализован каталог товаров с фильтрацией, корзина с сохранением состояния в `localStorage` и базовый Git-воркфлоу согласно заданию ПР-4.

## ✨ Функционал
- 📦 **Каталог товаров** с фильтрацией по категории и цене
- 🛒 **Корзина**: добавление, удаление, полная очистка
- 💾 **LocalStorage**: автосохранение корзины, данные не теряются при перезагрузке страницы
- 🧮 **Подсчет итоговой суммы** в реальном времени
- 🌐 Чистый HTML/CSS/JS без зависимостей

## 🛠 Технологии
- `HTML5`, `CSS3`, `Vanilla JavaScript (ES6+)`
- Web API: `localStorage`, `JSON.stringify()` / `JSON.parse()`
- `Git` & `GitHub` (ветвление, коммиты, merge)
